# Customer Segmentation Dashboard

A Power BI dashboard that analyzes customer demographics, spending behavior, income, purchase frequency, and product preferences. The project provides interactive visualizations and actionable insights to help businesses understand customer segments and improve marketing strategies.

## Tools Used
- Power BI
- Power Query
- DAX

## Key Features
- Customer Demographics Analysis
- Revenue & Spending Insights
- Product Category Performance
- Location-wise Analysis
- Interactive Dashboard Visualizations

## Outcome
Provides data-driven insights for customer targeting, business growth, and strategic decision-making.
